== Description ==

**WPshopping is an Open Source E-Commerce solution which offers a lot of Features. It is written in PHP and can be used in typical PHP/MySQL environments. WPshopping is used together with a WordPress.**

= Some of the standard Features are listed here: =

= General Features =
1. Shoppers can manage their User Accounts
2. Multiple Currencies
3. Multiple Payment Methods (Paypal, Nachnahme, Vorauskasse, Lastschrift, Sofortueberweisung).
4. Ability to add new Payment Methods
5. Product Ratings & Testimonials
6. Multilanguages
7. Manages an unlimited Number of Products and Categories.
8. Image for every Category.
9. Images, videos, demos can be used for each Product.
10. Different Attributes can be added to Product.
11. Order History (Customer can see all his orders)
12. WishList
13. Tax Management (for each Product can be different tax mode and for delivery too)
14. Delivery Management. Multiple ways of delivery (Standartversand, Express). Different price modes for different countries and depending on weight.
15. Ability to add new delivery methods.
16. Order Confirmation Mail is sent to Shopper and Store Owner
17. Different templates for Categories
18. Different templates for Products
19. Purchase without registering

= Administration Features =
1. Categories Management
2. Product Management
3. List of Orders
4. Manufacturers Management
5. Customers
6. Taxes Management
7. Order Status Management
8. Payments Methods
9. Delivery Methods
10. Prices for Delivery
11. Shopper Groups for Customers
12. Comments & Testimonials
13. Manage different Currencies, Countries
14. Import & Export

== Installation ==

= Minimum Requirements =

* WordPress 3.6 or greater
* PHP version 5.1.2 or greater
* MySQL version 4.0 or greater

= Manual installation =

The manual installation method involves downloading our WOPshop plugin and uploading it to `/wp-content/plugins/wopshop` directory via your FTP application. The WordPress codex contains [instructions on how to do this here](http://codex.wordpress.org/Managing_Plugins#Manual_Plugin_Installation).

1. Upload the plugin files to the `/wp-content/plugins/wopshop`, or install the plugin through the WordPress plugins screen directly.
1. Activate the plugin through the 'Plugins' screen in WordPress
1. Use the Settings->Plugin Name screen to configure the plugin
